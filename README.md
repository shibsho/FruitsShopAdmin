# FruitsShopAdmin

## Versions
- Python3.6.6
- Django2.1.4

## Quick Start

```sh
## install dependencies
$ pip install -r requirements.txt
## migrate
$ python manage.py migrate
## create admin user
$ python manage.py createsuperuser
## run local server
$ python manage.py runserver
## Access
$ open http://localhost:8000/
```
